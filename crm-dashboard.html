CRM Dashboard with AI Chatbot

A complete, production-ready CRM Dashboard built with Next.js, TypeScript, Prisma, and PostgreSQL

Live Demo GitHub Repository

About This Project

A comprehensive CRM (Customer Relationship Management) Dashboard that provides a complete solution for managing customer relationships, leads, deals, tasks, and activities. Built with modern web technologies, this application features role-based access control, real-time analytics, and an AI-powered chatbot for automated customer query handling.

The system includes full CRUD operations for leads and customers, a Kanban-style deal pipeline, comprehensive task management, activity logging, and detailed reporting with CSV export capabilities. The application is production-ready with secure authentication, database integration, and scalable architecture.

Tech Stack

Next.js 14 TypeScript PostgreSQL Prisma NextAuth.js Tailwind CSS Chart.js AI Chatbot

Key Features

Authentication & Security

Secure login/signup with NextAuth, role-based access control (Admin, Manager, Sales), and password hashing

Lead Management

Full CRUD operations with filtering, search, and conversion to customers

Customer Management

Complete customer profiles with history tracking, deals, and activities

Deal Pipeline

Kanban-style pipeline view for managing deals through stages

Task Management

Create and track tasks with due dates and assignments

Reports & Analytics

Comprehensive reports with sales performance, conversion rates, and CSV export

AI Chatbot

Automated customer query handling and support

User Management

Admin panel for managing users, roles, and permissions
